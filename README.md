This project demonstrates a robust Unsupervised Learning workflow using K-Means Clustering to extract meaningful patterns from an unlabeled dataset. The model's architecture was optimized by calculating the Within-Cluster Sum of Squares (WCSS) via the Elbow Method to identify the most efficient number of clusters. To mathematically validate the cohesion and separation of these clusters, I implemented Silhouette Analysis, ensuring high-quality data segmentation through a strong Silhouette Score. Finally, the model's reliability was confirmed by cross-referencing the predicted labels with a "ground truth" (labeled) dataset, achieving a high degree of accuracy and validating the clustering logic.
